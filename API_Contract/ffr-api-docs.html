FFR API — Financial Fitness Report (1.0.0)

Download OpenAPI specification:

Gemini-powered API that analyses structured financial data and scores across five pillars (spending, borrowing, protection, tax, wealth) to produce:

  • Pillar summaries — per-metric prose and a holistic overview with strengths and areas needing attention.
  • Insight cards — actionable, themed insight cards grouped by pillar with an optional cross-pillar top insight.

Both endpoints accept the same FfrScreenData payload and return structured JSON envelopes with consistent error shapes.

FFR

Financial Fitness Report endpoints — pillar summaries and insight cards.

Generate pillar summaries

Produces per-metric prose summaries and a holistic overall summary (strengths + areas needing attention) from structured financial data and pillar scores. metadata.type selects which pillars are in scope.

required

Structured financial data payload.
